## 主要公告（T-3 天，今日發佈）

**【維護預告】OpenStack 2026.1 升級**

本平台將於 **5/24（週日）21:00（UTC+8）** 進行 OpenStack 主版本升級（2025.2 → 2026.1），預計 150 分鐘內完成，最晚於同日 24:00 結束。

**維護期間暫停：**
- Horizon <https://openstack.cloudnative.tw>
- Skyline <https://console.cloudnative.tw>
- 所有 OpenStack API（含 `openstack` CLI、Terraform、CI/CD 自動化）

**維護期間不受影響：**
- 執行中的 instance 持續運作（不重啟、不遷移）
- Tenant network、Floating IP、Load Balancer 對外連線正常
- 已掛載的 Cinder volume、Swift 物件讀寫不中斷

**升級完成後一次性影響：**
- Horizon / Skyline 所有登入 session 失效（背景換成 Valkey），請重新登入。密碼不變。

**建議提前準備：**
- 暫停 5/24 21:00–24:00 之間的部署 / CI / Terraform apply。
- 編輯中的 Horizon 表單請先存草稿。
- 重要資料請維持平日備份習慣（本平台為社群實驗環境）。

即時進度會貼在本頻道；問題請寄 `infra@cloudnative.tw`。
